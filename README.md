### Third-party libraries

1. ##### less与less-loader：用于重定义（覆盖）第三方ui库样式

- less

```
npm install less --save-dev
```

- Less-loader:注意安装7.3.0版本，否则因为版本抬高，与vue版本不兼容而报错

```
npm i less-loader@7.3.0
```
